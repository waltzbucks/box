사전조건

설치를 시작하기 전에 sudo 권한이 있는 사용자로 로그인 했는지 확인 하십시오.

설치

OpenJDK 설치

  1. apt 패키지 인덱스 업데이트:
    $ sudo apt update

  1. OpenJDK 설치:
    $ sudo apt install -y default-jdk

  1. Java version 확인:
    $ java -version
    Output
openjdk version "1.8.0_265"
OpenJDK Runtime Environment (build 1.8.0_265-8u265-b01-0ubuntu2~16.04-b01)
OpenJDK 64-Bit Server VM (build 25.265-b01, mixed mode)

Tomcat 8.5 설치

  1. Tomcat user 생성:

    Tomcat 서비스를 실행 할 홈 디렉토리 /opt/tomcat 의 시스템 사용자 및 그룹 생성

    $ sudo useradd -m -U -d /opt/tomcat -s /bin/false tomcat

  1. Tomcat 다운로드:

    설치 및 다운로드에 필요한 패키지 설치

    $ sudo apt install -y unzip wget

    /tmp 디렉토리에 tomcat 을 다운로드

    $ cd /tmp
$ wget https://downloads.apache.org/tomcat/tomcat-8/v8.5.59/bin/apache-tomcat-8.5.59.zip

    * Tomcat 다운로드 링크: https://tomcat.apache.org/download-80.cgi

  1. Tomcat 설치:
    $ unzip apache-tomcat-*.zip
$ sudo mkdir -p /opt/tomcat
$ sudo mv apache-tomcat-8.5.59 /opt/tomcat/

  1. Symbolic link 생성:
    $ sudo ln -s /opt/tomcat/apache-tomcat-8.5.59/ /opt/tomcat/latest

  1. /opt/tomcat 디렉토리 권한 변경:
    $ sudo chown -R tomcat: /opt/tomcat

  1. bin 디렉토리 실행 권한 추가:
    $ sudo sh -c 'chmod +x /opt/tomcat/latest/bin/*.sh'

systemd 등록

Tomcat 을 service 로 등록하기위해 unit file tomcat.service 생성

  1. tomcat.service 생성
    $ cd /etc/systemd/system/
$ sudo vi /etc/systemd/system/tomcat.service

    /etc/systemd/system/tomcat.service

    [Unit]
Description=Tomcat 8.5 servlet container
After=network.target

[Service]
Type=forking

User=tomcat
Group=tomcat

Environment="JAVA_HOME=/usr/lib/jvm/default-java"
Environment="JAVA_OPTS=-Djava.security.egd=file:///dev/urandom"

Environment="CATALINA_BASE=/opt/tomcat/latest"
Environment="CATALINA_HOME=/opt/tomcat/latest"
Environment="CATALINA_PID=/opt/tomcat/latest/temp/tomcat.pid"
Environment="CATALINA_OPTS=-Xms512M -Xmx1024M -server -XX:+UseParallelGC"

ExecStart=/opt/tomcat/latest/bin/startup.sh
ExecStop=/opt/tomcat/latest/bin/shutdown.sh

[Install]
WantedBy=multi-user.target

  1. Tomcat service 실행:
    $ sudo systemctl daemon-reload
$ sudo systemctl start tomcat

  1. Tomcat 서비스 상태 확인:
    $ sudo systemctl status tomcat
    Output

  1. 부팅시 자동시작 설정:
    $ sudo systemctl enable tomcat

방화벽 설정

우분투 방화벽 ufw 에 tomcat service port 8080 허용 설정

$ sudo ufw allow 8080/tcp
