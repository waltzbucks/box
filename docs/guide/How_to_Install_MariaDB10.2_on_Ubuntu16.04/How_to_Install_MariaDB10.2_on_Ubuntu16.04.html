How to Install MariaDB 10.2 on Ubuntu 16.04

사전조건
MariaDB 10.2 설치

사전조건

설치를 시작하기 전에 sudo 권한이 있는 사용자로 로그인 했는지 확인 하십시오.

MariaDB 10.2 설치

현재 2020년 10월 15일 최종 버전은 10.5.6 입니다.

  1. 업데이트 및 필요 패키지 설치
    $ sudo apt update && sudo apt upgrade -y
$ sudo apt-get install -y software-properties-common gnupg-curl

  1. MariaDB repository 추가
    $ sudo apt-key adv --fetch-keys 'https://mariadb.org/mariadb_release_signing_key.asc'
$ sudo add-apt-repository 'deb [arch=amd64,arm64,i386,ppc64el] https://mirror.yongbok.net/mariadb/repo/10.2/ubuntu xenial main'

  1. MariaDB 설치
    $ sudo apt update
$ sudo apt install mariadb-server

    * 설치 진행시 "root" 패스워드 설정

  1. MariaDB 서비스 상태 확인
    $ sudo systemctl status mariadb
    Output

   Loaded: loaded (/lib/systemd/system/mariadb.service; enabled; vendor preset: enabled)

  1. MariaDB 서버 버전 확인
    $ mysql -V
    Output
mysql  Ver 15.1 Distrib 10.2.34-MariaDB, for debian-linux-gnu (x86_64) using readline 5.2

  1. MariaDB 접속 확인
    $ mysql -u root -p

    * 패스워드는 3. MariaDB 설치 단계에서 설정한 "root" 패스워드를 입력
